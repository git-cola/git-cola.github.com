Linux

Linux is it! Your distro has probably already packaged git-cola. If not, please file a bug against your distribution ;-)

arch » pacman -S git-cola
debian » apt-get install git-cola
fedora » yum install git-cola
gentoo » emerge cola
ubuntu » apt-get install git-cola

More? Contact the list if you've packaged git-cola for other distributions.

Mac OS X

Install git-cola using brew.

brew install git-cola

After installing git-cola using brew you can use the git-cola application bundle. Brew installs git-cola's dependencies (such as PyQt4) so you must install git-cola once using brew before downloading newer versions here.

Windows

Install the latest stable Python 2.6 binary packages listed below

  • git
  • python
  • pyqt4
  • git-cola
